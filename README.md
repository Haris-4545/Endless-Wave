# Thread — Endless Wave
 
A simple browser game. Navigate a dot through an endless series of gates as the wave accelerates.
 
**[Play it →](https://Haris-4545.github.io/endless-wave/thread-endless-wave.html)**
 
---
 
## How to play
 
The dot moves along a sine wave automatically. Hold any input to freeze it in place — release to let it go again.
 
| Input | Action |
|---|---|
| Click / Tap | Hold to pause the wave |
| Space | Hold to pause the wave |
 
Pass through the gaps to score. Every gate passed speeds the wave up slightly.
 
## Run locally
 
Just open the file:
 
```bash
git clone https://github.com/Haris-4545/endless-wave.git
cd endless-wave
open thread-endless-wave.html
```
 
## Built with
 
Vanilla HTML, CSS, and Canvas API — no dependencies.
